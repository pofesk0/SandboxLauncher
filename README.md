# SandboxLauncher

Это приложение главного экрана (лаунчер), которое позволяет скрывать приложения, а также при использовании спецвозможностей позволяет возвращать пользователя на главный экран при открытии приложения не через лаунчер, а также после выключения экрана. Это поможет ещё лучше защитить скрытые приложения.
